# 🛍️ Clothing Rental & E-Commerce Web Application

A web application for renting wedding clothing and 
selling regular apparel with full product management.

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL

## Features
- Product listing & management
- Rental management system
- Basic CRUD operations
- Responsive UI across devices
- Backend transactions with PHP & MySQL

## Setup
1. Clone the repo
2. Import `database.sql` into MySQL via phpMyAdmin
3. Configure `config.php` with your DB credentials
4. Run via XAMPP → localhost
